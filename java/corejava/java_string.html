String class

The String class represents character strings. All string literals in Java programs, such as "abc", are implemented as instances of this class.

Strings are constant,their values cannot be changed after they are created. String buffers support mutable strings.

String str = "abc";

is equivalent to:

char data[] = {'a', 'b', 'c'};

String str = new String(data);

Creating a String

There are two ways to create a string in Java:

i) String literal

String s = " BrightITCareer ";

ii) Using new keyword

String s = new String ( "BrightITCareer");

Methods of String

String Methods

int length():
Returns the number of characters in the String.

"BrightITCareer".length(); // returns 14

Char charAt(int i):
Returns the character at ith index.

"BrightITCareer".charAt(3); // returns 'g'

String substring (int i):
Return the substring from the ith index character to end.

"BrightITCareer".substring(3); // returns "ghtITCareer"

String substring (int i, int j):
Returns the substring from i to j-1 index.

"BrightITCareer".substring(2, 5); // returns "igh"

String concat( String str):
Concatenates specified string to the end of this string.

String s1 = "Bright";

String s2 = "ITCareer";

String output = s1.concat(s2); // returns "BrightITCareer"

int indexOf (String s):
Returns the index within the string of the first occurrence of the specified string.

String s = "Learn Share Learn";

int output = s.indexOf("Share"); // returns 6

int indexOf (String s, int i):
Returns the index within the string of the first occurrence of the specified string, starting at the specified index.

String s = "Learn Share Learn";

int output = s.indexOf('a',3);// returns 8

int lastindexOf( int ch):
Returns the index within the string of the last occurrence of the specified string.

String s ="Learn Share Learn";

int output = s.lastindexOf('a'); // returns 14

boolean equals( Object otherObj):
Compares this string to the specified object.

Boolean out = "Bright".equals("Bright"); // returns true

Boolean out = "Bright".equals("bright"); // returns false

boolean equalsIgnoreCase (String anotherString):
Compares string to another string, ignoring case considerations.

Boolean out="Bright".equalsIgnoreCase("Bright"); // returns true

Boolean out = "Bright".equalsIgnoreCase("bright"); // returns true

int compareTo( String anotherString):
Compares two string lexicographically.

int out = s1.compareTo(s2); // where s1 and s2 are

// strings to be compared

This returns difference s1-s2. If :

out < 0 // s1 comes before s2

out = 0 // s1 and s2 are equal.

out >0 // s1 comes after s2.

int compareToIgnoreCase( String anotherString):
Compares two string lexicographically, ignoring case considerations.

int out = s1.compareTo(s2); // where s1 ans s2 are

// strings to be compared

This returns difference s1-s2. If :

out < 0 // s1 comes before s2

out = 0 // s1 and s2 are equal.

out >0 // s1 comes after s2.

Note- In this case, it will not consider the case of a letter (it will ignore whether it is uppercase or lowercase).

String toLowerCase():
Converts all the characters in the String to lower case.

String word1 = "HeLLo";

String word3 = word1.toLowerCase(); // returns "hello"

String toUpperCase():
Converts all the characters in the String to upper case.

String word1 = "HeLLo";

String word2 = word1.toUpperCase(); // returns "HELLO"

String trim():
Returns the copy of the String, by removing whitespaces at both ends. It does not affect whitespaces in the middle.

String word1 = " Learn Share Learn ";

String word2 = word1.trim(); // returns "Learn Share Learn"

String replace (char oldChar, char newChar):
Returns new string by replacing all occurrences of oldChar with newChar.

String s1 = "feeksforfeeks";

String s2 = "feeksforfeeks".replace('f' ,'g'); // returns "geeksgorgeeks"
