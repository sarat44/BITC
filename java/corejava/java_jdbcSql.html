SQL Database Connection

Steps to interact with SQL Server database

  1. Download or find sqljdbc4-4.0.jar file
  2. We have to set the class path for the jar file
    1. My computer ->properties -> advanced system settings -> click on Environment Variables button -> click on new button from user variables


    Before Connecting, We need to install the SQL Server database.

  3. To Connect with database. we have to follow the below Steps.

Steps 1: import the package. 

import java.sql.*;
import javax.sql.*;

Steps 2: Load the driver. 

Class.forName("com.microsoft.sqlserver.jdbc.SQLServerDriver");

Steps 3: Establish the database connection. 

String url="jdbc:sqlserver://localhost:1433; databaseName=sarat; user= sa; password=sarat444@";
con = DriverManager.getConnection(url);

Steps 4: Create the statement object and send the commands to the database. 

Statement ps=con.CreateStatement();

Steps 5: Send the commands(Querys) and get the results by using the methods of Statement object. 

ResultSet rs=stmt.executeQuery(“select command”);
boolean flag=stmt.execute(“any  command”);
int i=stmt.executeUpdate(“only dml commands”);
int i[]=stmt.executeBatch();

Steps 6: Close the database connection 

con.close();

Example Code: 

import java.sql.*;
import static java.lang.System.*;
class DbCon
{	
	public static void main(String args[])throws Exception
	{	
			Class.forName("com.microsoft.sqlserver.jdbc.SQLServerDriver");
			System.out.println("hello driver...");
			String url="jdbc:sqlserver://localhost:1433; databaseName=sarat; user= sa; password=sarat444@";
	        Connection con = DriverManager.getConnection(url);
			Statement stmt=con.createStatement();
			ResultSet rs=stmt.executeQuery("select * from  emp");
			while(rs.next())
			{	
				for(int i=1;i<=4;i++)
				{
					out.print(rs.getString(i)+"\t");
				}
				out.println();
			}
		con.close();
	}
}
