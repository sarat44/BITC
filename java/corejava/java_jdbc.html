Jdbc Connection

Steps to interact with oracle database

  1. Download or find ojdbc14.jar/ojdbc6.jar file
  2. We have to set the class path for the jar file
My computer -> properties -> advanced system settings -> click on Environment Variables button -> click on new button from user variables


  1. Find out the SID Name of database
    1. Select INSTANCE_NAME from v$instance;
    2. Select name from v$database;

    Note: The above two commands are run only in administrator user (system/manager)

  2. What is 1521 port over Here?
    1. 1521 is the default port number of oracle. It allows oracle client connections to the database.
  3. To Connect with database. we have to follow the below Steps.

Steps 1: import the package. 

import java.sql.*;
import javax.sql.*;

Steps 2: Load the driver. 

Class.forName(“oracle.jdbc.driver.OracleDriver”);

Steps 3: Establish the database connection. 

Connection con=DriverManager.getConnection(“jdbc:oracle:thin:@PC3:1521:XE”,”scott”,”tiger”);

Steps 4: Create the statement object and send the commands to the database. 

Statement stmt=con.createStatement();

Steps 5: Send the commands(Querys) and get the results by using the methods of Statement object. 

ResultSet rs=stmt.executeQuery(“select command”);
boolean flag=stmt.execute(“any  command”);
int i=stmt.executeUpdate(“only dml commands”);
int i[]=stmt.executeBatch()

Steps 6: Close the database connection. 

con.close();

Example Code:

import java.sql.*;
import static java.lang.System.*;
class DbCon
{	
	public static void main(String args[])throws Exception
	{	Class.forName("oracle.jdbc.driver.OracleDriver");
		out.println("Loaded....");
		Connection con=DriverManager.getConnection("jdbc:oracle:thin:@localhost:1521:ORCL","scott","tiger");
		out.println("Connected...");
		Statement stmt=con.createStatement();
		ResultSet rs=stmt.executeQuery("select * from  emp");
		while(rs.next())
		{	for(int i=1;i<=4;i++)
			{out.print(rs.getString(i)+"\t");
			}
			out.println();
		}
		con.close();
	}
}

Note: Before that we need to have Database.
